# 《星轨便利店》美术规范

## 总体方向

- 类型：中文青春悬疑向 Galgame / Visual Novel
- 视觉关键词：夏夜、便利店冷光、旧天文馆、星轨、雨夜、毕业前夕
- 画风：日系视觉小说、赛璐璐半身立绘、柔和电影光影、干净线条

## 角色立绘规范

统一画布：

- 尺寸：832 x 1200
- 格式：WebP
- 背景：透明
- 构图：半身立绘，脚底/下缘对齐，角色中心稳定
- 表情：`neutral` / `smile` / `sad` / `surprised` / `serious` / `gentle`

角色差异：

| 角色 | 外观关键词 |
|------|------------|
| 星野未央 | 深紫长发、相机挂绳、安静眼神、星空色调 |
| 沈知夏 | 马尾、细框眼镜、校服、理性克制 |
| 白川音 | 白色连帽外套、浅发色、不对称刘海、神秘轻快 |
| 林母 | 围裙、温柔疲惫、暖色家庭感 |
| 周远 | 运动服、短发凌乱、少年感、热心朋友 |

当前仓库素材由 `scripts/generate-assets.mjs` 生成，保留统一轮廓与表情差异；后续若替换为外部 AI/手绘素材，只需保持相同路径和文件名。

## 背景规范

统一画布：

- 尺寸：1920 x 1080
- 格式：WebP
- 构图：16:9，重要视觉信息避开底部文本框区域
- 后处理：保留暗角和底部可读性渐变

场景列表：

- 深夜便利店
- 白天便利店
- 黄昏教室
- 旧天文馆
- 天台星空
- 清晨车站
- 雨夜商店街
- 地下星象仪
- 旧校舍走廊
- 傍晚居家
- 海边黄昏
- 标题星空

## 复现命令

```bash
npm run generate-assets
npm run download-audio
```

生成后素材会落在 `public/assets/`，由 `src/data/assets.ts` 统一引用。
